# **App Name**: PitchRating

## Core Features:

- Player Listing & Detail Pages: Browse a catalog of soccer players with dedicated profile pages displaying key stats and all submitted reviews.
- Player Review Submission: Authenticated users can submit detailed review articles for any soccer player, including a rating.
- Sponsored Ad Submission Form: Users can submit proposals and content for sponsored advertisements, which will be displayed on review list pages. Payment integration is a future enhancement for MVP.
- Admin Panel: Review Management: Administrators can view, approve, reject, edit, or delete submitted player review articles through a dedicated dashboard.
- Admin Panel: Ad Management: Administrators can manage (approve, reject, delete) sponsored advertisements, controlling their appearance on review list pages, from a central dashboard.
- AI Review Analysis Tool: An AI tool that processes multiple reviews for a player to generate a concise summary of key sentiment and discussion points.

## Style Guidelines:

- The application employs a dark color scheme, providing a modern and focused visual experience suitable for deep analysis. The primary action color is a vibrant blue, '#66A8F5' (RGB: 102, 168, 245), conveying professionalism and dynamism. The background is a very dark, subtle blue-gray, '#20242C' (RGB: 32, 36, 44), chosen to be easy on the eyes during extended reading sessions. An accent color, '#7AE9FF' (RGB: 122, 237, 255), is a bright aqua that provides high contrast for calls to action and important highlights.
- Headlines and titles use 'Space Grotesk', a sans-serif font, for its contemporary, analytical, and slightly techy feel. Body text, especially for player reviews, uses 'Inter', a neutral and highly readable sans-serif, ensuring clear and comfortable reading experiences for detailed articles.
- Crisp, modern line icons are utilized throughout the application. These include soccer-related imagery (e.g., soccer balls, goal outlines), review indicators (e.g., star ratings, up/down arrows, speech bubbles), and administrative controls (e.g., edit, delete, approve icons) to maintain visual consistency and clarity.
- A content-first, responsive layout is employed, optimizing the viewing experience across devices. Player profiles feature a clear hierarchy with prominent display of player information, review sections, and interactive elements. Administrative dashboards adopt a structured, table-based approach for efficient management of articles and ads.
- Subtle and functional animations enhance user interaction. This includes smooth hover states for clickable elements like player cards and review snippets, as well as elegant fade-in/fade-out transitions for modal dialogs and navigation menus, contributing to a polished and professional user experience.
